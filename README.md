# Dotnet_WebApplicationTemplate

A starter template for modern, scalable .NET web applications using ASP.NET Core.  
This repository provides a solid base for building web APIs, Razor Pages, MVC apps, or hybrid solutions with best-practice structure and configuration.

---

## Overview

`Dotnet_WebApplicationTemplate` is a flexible and opinionated project template designed to accelerate the creation of new .NET web applications. It includes sensible defaults, a clean folder structure, and common middleware so developers can focus on application logic rather than boilerplate setup.

The template is suitable for both individual developers and team-based projects that require consistency and maintainability.

---

## Features

- Clean, readable project structure
- Support for ASP.NET Core Web API, MVC, Razor Pages, or Minimal APIs
- Built-in dependency injection configuration
- Environment-specific configuration files
- Common middleware preconfigured (routing, logging, error handling)
- Extensible architecture suitable for production systems
